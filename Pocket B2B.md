### 📱 About the App 

<br/>

<div align="center">
<p>Pocket B2B is a powerful wholesale ecosystem designed specifically to bridge the gap between large-scale grocery distributors and retail business customers. Built with two distinct modules, it provides a native mobile <b>User App</b> for retail store owners to seamlessly place bulk orders, and a comprehensive responsive <b>Admin Web Panel</b> for the distributors to oversee the entire B2B operation from a single vantage point.</p>
<br/>
<p>At its core, this architecture unifies complex wholesale workflows into one Flutter codebase leveraging GetX for reactive state and Firebase for real-time synchronization. Store owners experience frictionless bulk checkouts with Hive local caching and Razorpay integrations, while administrators gain complete control over inventory, order fulfillment pipelines, dynamic pricing, and fully automated PDF invoicing.</p>
</div>

### 🎯 Why This App Stands Out

<br/>

- 🏗️ **Adaptive Dual-Module Codebase** — Uses complex runtime checks and `flutter_screenutil` logic to render mobile views for retail stores on small devices, while spinning up powerful metrics-heavy dashboard web layouts for admins automatically based on device footprint.
- 💼 **Enterprise B2B Scalability** — Specifically engineered for wholesale scenarios with features like targeted bulk promotions, offline-ready persistent cart states (`hive_flutter`), and robust state management utilizing GetX.
- 🎨 **Data Visualization Engine** — The Admin Dashboard integrates `fl_chart` to provide distributors with an immediate, deep-dive view into real-time transactional statistics, stock health, and user metrics.
- ⚡ **Seamless Financial Workflows** — A comprehensive order pipeline starting from checkout mapping integrated highly securely via `razorpay_flutter`, terminating in automated enterprise-styled receipt generation utilizing `pdf` and `printing` modules.
- 🔄 **Real-Time Data Parity** — Immediate synchronization mechanism engineered back-to-front by pairing efficient Firestore data streaming with reactive GetX widgets allowing changes made in the Admin web portal to reflect in retail consumer applications in real-time.
